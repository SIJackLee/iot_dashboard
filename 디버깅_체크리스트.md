## 예상 에러 및 해결

### 1. CSS 변수 에러
**증상:** 색상이 표시되지 않음
**해결:** globals.css의 CSS 변수 확인

### 2. 컴포넌트 import 에러
**증상:** "Cannot find module" 에러
**해결:** 컴포넌트 경로 확인 (`@/components/ui/...`)

### 3. Tailwind 클래스 에러
**증상:** 스타일이 적용되지 않음
**해결:** tailwind.config.ts 확인 (shadcn/ui가 자동 설정)

### 4. 반응형 작동 안 함
**증상:** 모바일에서도 4열로 표시
**해결:** 브라우저 캐시 클리어, Tailwind 클래스 확인

## 알려진 이슈

### 1. Middleware 경고
- Next.js 16에서 middleware 파일 convention이 deprecated됨
- 현재는 동작에 문제 없음
- 추후 `proxy`로 마이그레이션 가능

### 2. next-themes 설치됨
- Sonner가 자동으로 설치했지만 사용하지 않음
- 제거 가능하지만 다른 컴포넌트에서 사용할 수도 있으므로 유지
